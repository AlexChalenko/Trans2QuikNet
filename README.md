# Trans2QuikNET

Trans2QuikNet- это C# библиотека для интеграции с торговой системой QUIK с использованием нативной библиотеки Trans2Quik.dll.

Ключевой особенностью является поддержка подключений к нескольким терминалам QUIK одновременно.

Библиотека находиться в разработке. Реализовано:
- [x] Подключение к терминалу QUIK
- [x] Отправка транзакций
- [x] Получение информации о заявках
- [ ] Получение информации о сделках
